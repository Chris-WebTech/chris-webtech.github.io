About The Project

The form utilizes method POST and stors the user inputs in appropreate session variable using $_SESSION; the data is assigned to the Associative Array membersArray which is transferable across webpages, this allows for access and minipulation on other webpages.

The Array Key and Values are coming from the name of the html field and the user inputs.
